# 🕹️ SHOT! — Pygame 슈팅 게임

간단한 2D 비행기 슈팅 게임입니다.  
Python의 **Pygame** 라이브러리를 이용해 제작되었으며,  
플레이어의 조작, 적 생성, 총알 발사, HP바, 점수 시스템 등을 모두 포함하고 있습니다.

---

## 🎮 주요 기능

| 기능              | 설명                                         |
| ----------------- | -------------------------------------------- |
| **인트로 화면**   | `Enter` 키로 게임 시작                       |
| **플레이어 조작** | 방향키로 이동, `Space` 키로 총알 발사        |
| **적 생성**       | 랜덤 위치에서 등장하며 주기적으로 총알 발사  |
| **충돌 판정**     | 총알 ↔ 적 / 적의 총알 ↔ 플레이어             |
| **HP 시스템**     | 플레이어 HP가 0이 되면 `GAME OVER`           |
| **점수 시스템**   | 적을 처치하면 +100점                         |
| **재시작 / 종료** | `R` 키로 재시작, `ESC` 또는 창 닫기로 종료   |
| **자동 적 생성**  | 3초마다 새 적 스폰 (`pygame.time.set_timer`) |

## 💡 주요 시스템 요약

### 🔸 HP 시스템

- `self.max_hp` = 100
- `self.current_hp` 감소 시 HP 바 색상 및 수치 반영
- HP가 0이 되면 `game_over = True`

### 🔸 점수 시스템

- 적이 3회 피격될 때마다 `enemy.kill()`
- `self.score += 100`

### 🔸 적 생성

- 시작 시 3개 랜덤 생성
- 이후 3초마다 자동 추가 (`pygame.time.set_timer` 사용)

## 🧱 개선 포인트 / 참고 사항

- 현재 총알 이미지와 적 크기, 이동 속도 등은 코드 내에서 직접 조정 가능
- `Enemy.update()`에서 총알의 `update()`는 호출하지 않아야 함  
  (중복 업데이트로 인해 총알 속도가 달라지는 현상 방지)
- 필요 시 배경음악, 폭발 애니메이션, 보스 스테이지 등을 확장 가능
